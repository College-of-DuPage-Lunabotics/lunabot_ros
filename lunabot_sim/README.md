# lunabot_sim

This package contains models and world files for simulation in Gazebo.

## Content
- **models**: Gazebo simulation models.
- **worlds**
  - **high_resolution**
    - **artemis**: artemis_arena.world, artemis_arena2.world, artemis_arena3.world, artemis_no_walls.world
    - **ucf**: ucf_arena.world
  - **low_resolution**
    - **artemis**: artemis_arena.world, artemis_arena2.world, artemis_arena3.world
