# lunabot_msgs

This package contains custom messages, actions, and services.

## Actions
- **Depositing.action**: Defines the depositing action.
- **Excavation.action**: Defines the excavation action.

## Messages
- **ControlState.msg**: Robot control state.
- **PowerMonitor.msg**: Power monitoring data.

## Services
- **LaunchSystem.srv**: Triggers launch of a system component on the robot.
- **StopSystem.srv**: Stops a running system component on the robot.
